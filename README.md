# Flappy Bird Web Game

A full-featured Flappy Bird game built with HTML, JavaScript, Phaser.js, and Canvas.

## How to Play

- Click or press Space to make the bird jump.
- Avoid the pipes and the ground.
- Score points by passing through pipes.

## Features

- 100% browser-based
- Built on HTML5 Canvas with Phaser.js
- Smooth animations and physics
- Sound effects (jump, score, game over)
- Responsive design
- Score tracking
- Restart functionality

## Technologies Used

- HTML
- JavaScript
- Phaser.js (game framework)
- Canvas API
- Web Audio API (for sounds)

## Running the Game

Open `index.html` in your web browser to start playing. No server required.
